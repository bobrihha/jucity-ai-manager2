# Knowledge base

Структура папок — по темам. Внутри каждой темы держим факты, правила и готовые формулировки для материалов/скриптов.

Разделы:
- `about_park` — что это за парк, формат, ключевые отличия
- `tickets` — билеты, тарифы, что входит/не входит, возвраты
- `food` — еда/напитки, ограничения, аллергены
- `activities` — активности, расписание, возрастные ограничения
- `birthdays` — дни рождения, пакеты, условия
- `graduation` — выпускные, форматы, условия
- `rules` — правила посещения, безопасность, запреты
- `logistics` — как добраться, парковка, время работы, контакты
