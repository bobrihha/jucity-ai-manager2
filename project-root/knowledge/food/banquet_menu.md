# Банкетное меню и угощения на праздники

## Суть
Для дней рождения и выпускных в парке можно заказать банкетное меню — готовые наборы и блюда для детей и взрослых.

## Факты
- Банкетное меню заказывается через ресторан парка
- Есть готовые детские наборы для праздников
- Можно заказать пиццы, горячие блюда, закуски и напитки
- Доступны специальные сеты для детских мероприятий
- Меню подбирается под количество гостей

## Важные нюансы
- Банкетное меню рекомендуется заказывать заранее
- Для праздников можно согласовать состав и количество блюд
- Свой торт обычно допускается, остальные угощения — через ресторан
- Алкоголь на детские праздники запрещён

## Как объяснять гостю
Для праздника можно заказать готовое меню прямо у нас — так удобнее и не нужно ничего везти с собой. Менеджеры помогут подобрать блюда под вашу компанию.
