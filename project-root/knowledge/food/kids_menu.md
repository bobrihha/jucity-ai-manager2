# Детское меню

## Суть
В ресторане парка есть отдельное детское меню — блюда подобраны специально для детей по составу и вкусу.

## Факты
- В меню есть блюда для детей разного возраста
- Предлагаются простые и понятные позиции: пюре, котлеты, наггетсы, паста, пицца
- Есть десерты и напитки для детей
- Порции адаптированы под детский аппетит

## Важные нюансы
- Детское меню подходит даже для самых маленьких гостей
- Блюда готовятся на месте
- При праздниках детское меню можно заказать заранее на группу

## Как объяснять гостю
Для детей у нас есть отдельное меню — простое, вкусное и понятное. Обычно дети с удовольствием едят после активных игр.
