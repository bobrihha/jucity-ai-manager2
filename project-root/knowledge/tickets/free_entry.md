# Бесплатный вход

## Суть
В парке входной билет требуется только для детей. Сопровождающие взрослые проходят бесплатно.

## Факты
- Взрослые сопровождающие проходят бесплатно
- Билет покупается только для ребёнка
- Дети до 1 года также посещают парк бесплатно
- Ограничений по количеству взрослых нет

## Важные нюансы
- Взрослые могут находиться в парке вместе с детьми без доплаты
- Взрослые сопровождают детей, но не пользуются аттракционами
- Для детей до 1 года может потребоваться документ, подтверждающий возраст

## Как объяснять гостю
Билет нужен только для ребёнка. Родители и другие взрослые могут быть рядом и находиться в парке бесплатно — это уже включено.
